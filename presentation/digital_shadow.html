01Сбор данных

Коллекторы (фреймворк)

  • FileCollector — JSONL экспортированных листингов (надёжно, легально)
  • HttpPageCollector — публичные страницы (httpx); .onion через --proxy socks5h://…
  • RssCollector — RSS/Atom-ленты (stdlib xml)
  • DarknetMock — синтетика для демо
  • CLI collect.py: источник → пайплайн → граф+БД → топ угроз
§0: только публичный текст, без контактов/покупок, без хранения ПДн. Наркотики/реальный даркнет — синтетика/research, не краулинг.

03Скоринг и приоритизация

Детерминированный риск + приоритет

shadow_score(signals):
  w = шадоу-вес или core-вес
  risk = min(100, Σ w(s))
  level = low/medium/high/critical

prioritize():
  threat = risk × confidence × source + reuse
  confidence ≤ 1.2  (рост с числом сигналов)
  source: darknet 1.0 · paste 0.9 · clearweb 0.75

priority

0–34low
35–59medium
60–84high
85+urgent
Один и тот же набор сигналов из даркнета приоритетнее, чем из clearweb.
04Главная ценность

Общий граф — мост между продуктами

Источники различаются лейблом (:Video/:Post/:Call у Media, :ShadowItem у Shadow), а сущности — общие узлы: :Domain · :Wallet · :TelegramUsername · :PromoCode.

Кошелёк из TikTok-скама и из даркнет-листинга = один узел → видна общая теневая сеть. entity_reuse по name|code|address → сигнал graph_entity_reuse.

Проверено вживую: общий кошелёк → reuse=2, в окрестности и Video, и ShadowItem.
(:Video)      ─[:MENTIONS]─┐
                            ▼
                       (:Wallet)  ← один узел
                            ▲
(:ShadowItem) ─[:MENTIONS]─┘

# кросс-продуктовый запрос
MATCH (e)<-[:MENTIONS]-(s)
WHERE 'ShadowItem' IN kinds
  AND 'Video' IN kinds
RETURN e, uses

§0Правила

Безопасность данных

✕ Нельзя

  • Контакт/покупка у продавцов
  • Хранить реальные ПДн из утечек (ИИН/карты/телефоны)
  • Краулить реальные даркнет/наркоплощадки
  • Выводы «этот человек — преступник»

✓ Можно

  • Публичный текст/метаданные объявлений
  • Публичные abuse-репорты (кошельки)
  • Синтетика для наркотиков/даркнета
  • Только маски/хэши реквизитов; вывод — risk scoring
